# Instruction-rule decisions log

Closed judgments from the instruction-review application (`#ai-instruction-review`): proposals that were **rejected**, **folded** into another rule, or **deferred** until a condition holds. One line per `#tag`, no draft blocks.

This is the round's only committed output. Open proposals and their drafts are **ephemeral** (regenerated by each full audit, dispositioned in triage, never committed); only these decisions persist, so future rounds neither re-litigate a rejection nor prematurely re-raise a deferred rule. Dropped drafts remain recoverable from git history.

A deferred entry carries a placement hint `(-> <targetFile>, <role>)` so a future round, on the condition holding, re-drafts in the right place.
